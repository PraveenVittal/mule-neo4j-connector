Neo4J Connector
=============================

The Anypoint Connector for Neo4J is a closed-source, SDK-based connector that provides automation of processes for Neo4J operations.

## Author
MuleSoft Inc.

## Supported Mule runtime versions
Mule 3.7.x, 3.8.x

## Neo4J Bolt API supported versions
Neo4J API 1.2.1

## Supported modules

* CQL
* Nodes
